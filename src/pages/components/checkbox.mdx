## Sizes

Los checkbox se pueden mostrar en diferentes tamaños. Por defecto, el tamaño es `md`. Los tamaños disponibles son `sm`, `md`, `lg` y `xl`. Para cambiar el tamaño, se debe usar la propiedad `size`.

<Example>
  <div class="flex items-center gap-6">
    <Checkbox size="sm" />
    <Checkbox size="md" />
    <Checkbox size="lg" />
    <Checkbox size="xl" />
  </div>
</Example>

## Variants

Los checkbox se pueden mostrar en diferentes variantes. Por defecto, la variante es `primary`. Las variantes disponibles son `primary`, `secondary`, `success`, `error`, `light` y `dark`. Para cambiar la variante, se debe usar la prop `variant`.

<Example>
  <div class="flex items-center gap-6">
    <Checkbox variant="primary" />
    <Checkbox variant="secondary" />
    <Checkbox variant="success" />
    <Checkbox variant="error" />
    <Checkbox variant="dark" />
  </div>
</Example>

```astro
<Checkbox variant="primary" />
<Checkbox variant="secondary" />
<Checkbox variant="success" />
<Checkbox variant="error" />
<Checkbox variant="dark" />
```

## Props

| Prop | Type | default | Description |
| ---- | ---- | ------- | ----------- |
| size | String | md | Esta prop permite cambiar el tamaño del checkbox. Los tamaños disponibles son `sm`, `md`, `lg` y `xl`. |
| variant | String | primary | Esta prop permite cambiar la variante del checkbox. Las variantes disponibles son `primary`, `secondary`, `success`, `error`, `light` y `dark`. |
| light | Boolean | false | Esta prop permite cambiar el modo del checkbox a `light`. |
| checked | Boolean | false | Esta prop permite cambiar el modo del checkbox a `checked`. |
| disabled | Boolean | false | Esta prop permite cambiar el modo del checkbox a `disabled`. |
