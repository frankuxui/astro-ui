## Usage

Para dar funcionalidad a los modales, se debe usar el atributo `data-toggle` en un elemento que desencadene el modal. El valor del atributo `data-toggle` debe ser el `id` del modal que se quiere mostrar.

<Example>
  <Button data-toggle='modal-example'>Example</Button>
  <Modal id='modal-example'>
    <h2 slot='modal-title' class='modal-title'>Modal title</h2>
    <p slot='modal-body'>Lorem ipsum dolor sit amet, consectetur adipiscing...</p>
    <Button slot='modal-footer' type='button' data-dismiss='modal' variant='secondary'>Close</Button>
    <Button slot='modal-footer' type='button' variant='primary'>Save changes</Button>
  </Modal>
</Example>

## Placements

Los modales se pueden mostrar en diferentes posiciones. Por defecto, la posición es `top-center` y las posiciones disponibles son `top-start`, `top-center`, `top-end`, `center-start`, `center`, `center-end`, `bottom-start`, `bottom-center` y `bottom-end`. Para cambiar la posición, se debe usar la propiedad `placement`.

## Static backdrop

Los modales se pueden mostrar con un fondo estático. Para cambiar el modo, se debe usar la propiedad `staticBackdrop`.

<Example>
  <Button data-toggle='modal-example-static'>Static backdrop</Button>
  <Modal id='modal-example-static' staticBackdrop>
    <h2 slot='modal-title' class='modal-title'>Modal static backdrop</h2>
    <p slot='modal-body'>Lorem ipsum dolor sit amet, consectetur adipiscing...</p>
    <Button slot='modal-footer' type='button' data-dismiss='modal' variant='secondary'>Close</Button>
    <Button slot='modal-footer' type='button' variant='primary'>Save changes</Button>
  </Modal>
</Example>

## Modal content

Este ejemplo es fundamental para si queremos personalizar el contenido del modal. En este caso, el modal no tiene un `modal-body`, ni un `modal-header` ni un `modal-footer`. En su lugar, tiene un `modal-content` que es el que contiene todo el contenido del modal.

Una sugerencia para aplicar estilos es el uso de [Tailwind CSS](https://tailwindcss.com/).

<Example class='example-modal-docs'>
  <Modal>
    <div slot='modal-content'>
      <h2 class='modal-title'>Modal title</h2>
      <p>Lorem ipsum dolor sit amet, consectetur adipiscing...</p>
      <Button type='button' variant='secondary'>Close</Button>
      <Button type='button' variant='primary'>Save changes</Button>
    </div>
  </Modal>
</Example>

```astro
<Modal>
  <div slot='modal-content' class='custom-tailwind-class'>
```
